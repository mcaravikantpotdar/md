# Unit-1-Fundamentals of Computer / कंप्यूटर के मूल सिद्धांत

### What is Computer? Define Computer. / कंप्यूटर क्या है? कंप्यूटर को परिभाषित करें।

| English | हिंदी |
| :--- | :--- |
| Computer is an electronic device. | कंप्यूटर एक इलेक्ट्रॉनिक उपकरण है। |
| It takes data and instructions as input. | यह data और instructions को input के रूप में लेता है। |
| It processes the data according to the given instructions and gives information as output. | यह दिए गए निर्देशों के अनुसार data को process करता है और output के रूप में information देता है। |

---

### Block Diagram of Computer / कंप्यूटर का ब्लॉक डायग्राम



| English | हिंदी |
| :--- | :--- |
| **Input Unit:** The computer receives data and instructions from the input unit. | **Input Unit:** कंप्यूटर input unit से data और instructions प्राप्त करता है। |
| The input unit changes the data and instructions from human language to binary language. | यह unit डेटा और निर्देशों को human language से binary language में बदलता है। |
| All the input devices, like the keyboard and mouse, are connected to the input unit of the computer. | कीबोर्ड और माउस जैसे सभी input devices कंप्यूटर की input unit से जुड़े होते हैं। |
| **Output Unit:** The computer provides information from the Output Unit. | **Output Unit:** कंप्यूटर output unit से information प्रदान करता है। |
| The output unit changes the information from binary language to human language. | यह unit information को binary language से human language में बदलता है। |
| All the output devices, like the monitor and printer, are connected to the output unit of the computer. | मॉनिटर और प्रिंटर जैसे सभी output devices कंप्यूटर की output unit से जुड़े होते हैं। |
| **CPU:** The full form of CPU is Central Processing Unit. | **CPU:** CPU का full form Central Processing Unit है। |
| It is also called the brain of the computer. | इसे कंप्यूटर का brain भी कहा जाता है। |
| All the processing of data is done by the CPU. | डेटा की सारी processing CPU द्वारा की जाती है। |
| There are two main parts in a CPU: | CPU में दो मुख्य भाग होते हैं: |
| 1. **ALU:** The full form of ALU is Arithmetic & Logic Unit. | 1. **ALU:** ALU का full form Arithmetic & Logic Unit है। |
| It does all the mathematical and logical processing of the data. | यह data की सारी mathematical और logical processing करता है। |
| 2. **CU:** The full form of CU is Control Unit. | 2. **CU:** CU का full form Control Unit है। |
| It is also called the heart of the computer. | इसे कंप्यूटर का heart भी कहा जाता है। |
| It controls all the other units of the computer system. | यह computer system की सभी अन्य units को control करता है। |
| **MU:** Memory Unit is used to store data, instructions, and information in the computer. | **MU:** Memory Unit का उपयोग कंप्यूटर में data, instructions और information को store करने के लिए किया जाता है। |

### Types of Memories in a Computer / कंप्यूटर में मेमोरी के प्रकार

| English | हिंदी |
| :--- | :--- |
| There are two types of memories in a computer. | कंप्यूटर में दो प्रकार की memories होती हैं। |
| 1. **Primary Memory (Main Memory):** Primary memory is directly connected with the CPU. | 1. **Primary Memory (Main Memory):** Primary memory सीधे CPU से जुड़ी होती है। |
| It is very fast as compared to Secondary memory. | Secondary memory की तुलना में यह बहुत तेज़ होती है। |
| **ROM** is used to store some special instructions that help the computer start. | **ROM** का उपयोग कुछ विशेष instructions को store करने के लिए किया जाता है जो कंप्यूटर को start होने में मदद करते हैं। |
| ROM is a permanent memory and never loses its stored data. | ROM एक permanent memory है और यह अपना stored data कभी नहीं खोता है। |
| **RAM** is used to temporarily store data and instructions for currently running programs. | **RAM** का उपयोग वर्तमान में चल रहे programs के लिए data और instructions को temporarily store करने के लिए किया जाता है। |
| RAM is a volatile memory and loses its stored data after a power reset. | RAM एक volatile memory है और power reset के बाद यह अपना stored data खो देता है। |
| 2. **Secondary Memory (Auxiliary Memory):** Secondary memory is not directly connected with the CPU. | 2. **Secondary Memory (Auxiliary Memory):** Secondary memory सीधे CPU से जुड़ी नहीं होती है। |
| It is a non-volatile memory used to store data, instructions, and information permanently in the computer. | यह एक non-volatile memory है जिसका उपयोग कंप्यूटर में data, instructions और information को permanently store करने के लिए किया जाता है। |
| HDD, SSD, CD, and DVD are some examples of secondary memories. | HDD, SSD, CD और DVD सेकेंडरी मेमोरी के कुछ उदाहरण हैं। |

### What are the units of memory? / मेमोरी की इकाइयाँ क्या हैं?

| English | हिंदी |
| :--- | :--- |
| Memory units are used to measure and represent data. | Memory units का उपयोग data को measure और represent करने के लिए किया जाता है। |
| **Bit:** It is the smallest memory unit to measure data stored in memory. | **Bit:** यह मेमोरी में stored data को measure करने की सबसे छोटी unit है। |

* **1 Bit** = Value of 0 or 1
* **1 Nibble** = 4 bits
* **1 Byte (B)** = 8 bits
* **1 Kilobyte (KB)** = 1024 bytes
* **1 Megabyte (MB)** = 1024 KB
* **1 Gigabyte (GB)** = 1024 MB
* **1 Terabyte (TB)** = 1024 GB

### What do you mean by Input-Devices? Explain some Input Devices. / इनपुट-डिवाइस से आपका क्या अभिप्राय है? कुछ इनपुट डिवाइस के बारे में बताइए।

| English | हिंदी |
| :--- | :--- |
| Input devices are those which are connected to the input unit of the computer. | Input devices वे होते हैं जो कंप्यूटर की input unit से जुड़े होते हैं। |
| Input devices are used to provide data and instructions to the computer. | Input devices का उपयोग कंप्यूटर को data और instructions प्रदान करने के लिए किया जाता है। |

1. **Keyboard:** It is the most common input device used with computers. It is used to enter alphabets, numbers, symbols and special instructions into the computer.
*कीबोर्ड: यह कंप्यूटर के साथ उपयोग किया जाने वाला सबसे आम input device है। इसका उपयोग कंप्यूटर में alphabets, numbers, symbols और विशेष instructions दर्ज करने के लिए किया जाता है।*

2. **Mouse:** It is the most common pointing device used with computers. It is used to control the pointer-arrow and cursor on the screen.
*माउस: यह कंप्यूटर के साथ उपयोग किया जाने वाला सबसे आम pointing device है। इसका उपयोग स्क्रीन पर pointer-arrow और cursor को control करने के लिए किया जाता है।*

3. **Scanner:** It is used to make a digital copy of paper documents. It scans the paper document and stores its digital copy in the computer. There are three types of scanners: (i) Flat-bed Scanner, (ii) Drum Scanner, (iii) Line-Scanner or Hand-held Scanner.
*स्कैनर: इसका उपयोग कागज़ के दस्तावेज़ों की digital copy बनाने के लिए किया जाता है। यह कागज़ के दस्तावेज़ को scan करता है और इसकी digital copy कंप्यूटर में store करता है। स्कैनर तीन प्रकार के होते हैं: (i) Flat-bed Scanner, (ii) Drum Scanner, (iii) Line-Scanner या Hand-held Scanner।*

4. **Digital Camera:** It is used to take photos and videos and store them into the computer. Digital camera is used for video conferencing.
*डिजिटल कैमरा: इसका उपयोग photos और videos लेने और उन्हें कंप्यूटर में store करने के लिए किया जाता है। Digital camera का उपयोग video conferencing के लिए किया जाता है।*

5. **Joy-stick:** Joy stick is also a pointing device used for playing video games. It has a stick attached to a base that can move in all directions like up-down-right-left.
*जॉय-स्टिक: Joy stick भी एक pointing device है जिसका उपयोग video games खेलने के लिए किया जाता है। इसके base पर एक stick लगी होती है जो up-down-right-left सभी दिशाओं में घूम सकती है।*

6. **Touch-Screen:** It is an input device as well as an output device. We can give input by touching different areas and at the same time we can see the information on it.
*टच-स्क्रीन: यह एक input device होने के साथ-साथ एक output device भी है। हम स्क्रीन पर अलग-अलग क्षेत्रों को छूकर input दे सकते हैं और साथ ही उस पर information देख सकते हैं।*

7. **Light pen (or stylus):** It is a pen-shaped input device that is used to draw or write directly on the computer screen. Light-pen is used on CRT Monitors and Stylus is used on LCD Monitors.
*लाइटपेन (या स्टाइलस): यह एक pen के आकार का input device है जिसका उपयोग कंप्यूटर स्क्रीन पर सीधे draw या write करने के लिए किया जाता है। Light-pen का उपयोग CRT Monitors पर किया जाता है और Stylus का उपयोग LCD Monitors पर किया जाता है।*

8. **BCR:** Its full-form is Bar Code Reader. It is used to scan bar-codes, which are information stored in a set of lines to identify items in stores.
*BCR: इसका full-form Bar Code Reader है। इसका उपयोग bar-codes को scan करने के लिए किया जाता है। Bar-codes लाइनों के सेट में stored information होती है जिसका उपयोग स्टोर्स में items की पहचान करने के लिए किया जाता है।*

9. **MICR:** Its full-form is Magnetic Ink Character Reader. It is used in Banks to read information printed with Magnetic Ink.
*MICR: इसका full-form Magnetic Ink Character Reader है। इसका उपयोग Banks में Magnetic Ink से प्रिंट की गई information को पढ़ने के लिए किया जाता है।*

10. **OCR:** Its full form is Optical Character Recogniser. It recognizes text from a scanned document and makes an editable digital copy.
*OCR: इसका full form Optical Character Recogniser है। यह scan किए गए दस्तावेज़ से text को पहचानता है और एक editable digital copy बनाता है।*

### What do you mean by Output-Devices? Explain some Output Devices. / आउटपुट-डिवाइस से आपका क्या अभिप्राय है? कुछ आउटपुट डिवाइस के बारे में बताइए।

| English | हिंदी |
| :--- | :--- |
| Output devices are those which are connected to the output unit of the computer. | Output devices वे होते हैं जो कंप्यूटर की output unit से जुड़े होते हैं। |
| Output devices are used to get information from the computer. | Output devices का उपयोग कंप्यूटर से information प्राप्त करने के लिए किया जाता है। |

1. **Monitor (VDU):** Monitor is the most commonly used output device, also known as VDU (Visual Display Unit). It looks like a TV set and information displayed on it is called "soft-copy". Monitors are of two types: (i) CRT, (ii) LCD.
*मॉनिटर (VDU): Monitor सबसे अधिक इस्तेमाल किया जाने वाला output device है, जिसे VDU (Visual Display Unit) के नाम से भी जाना जाता है। यह एक TV सेट की तरह दिखता है और इस पर प्रदर्शित होने वाली information को "soft-copy" कहा जाता है। मॉनिटर दो प्रकार के होते हैं: (i) CRT, (ii) LCD।*

2. **Printer:** It is an output device used to print information on paper, which is known as "Hard-copy". Printers are divided into two types:
*प्रिंटर: यह एक output device है जिसका उपयोग कागज़ पर information को प्रिंट करने के लिए किया जाता है, जिसे "Hard-copy" कहा जाता है। प्रिंटर दो प्रकार के होते हैं:*
    * **(i) Impact Printers:** The printing mechanism directly presses against the paper. They are very noisy, slow, and inexpensive. Examples: Dot-Matrix Printer (character printer), Chain Printer (line printer), Drum Printer (page-printer).
        *(i) Impact Printers: इनमें printing mechanism सीधे कागज़ पर दबाव डालता है। ये बहुत noisy, धीमे और सस्ते होते हैं। उदाहरण: Dot-Matrix Printer (character printer), Chain Printer (line printer), Drum Printer (page-printer)।*
    * **(ii) Non-Impact Printers:** The printing mechanism does not press against the paper. They are less noisy, faster, expensive, and have high print quality. Examples: Inkjet Printer (line printer), Laser Printer (page printer).
        *(ii) Non-Impact Printers: इनमें printing mechanism कागज़ पर दबाव नहीं डालता है। ये कम noisy, तेज़, महंगे होते हैं और इनकी प्रिंट क्वालिटी अच्छी होती है। उदाहरण: Inkjet Printer (line printer), Laser Printer (page printer)।*

3. **Plotter:** It is used to draw high quality designs on large sheets of paper or canvas, like designs of buildings, ships, and aircraft. There are two types: (i) Flat-bed plotter, (ii) Drum Plotter.
*प्लॉटर: इसका उपयोग कागज़ या कैनवास की बड़ी शीट पर high quality designs बनाने के लिए किया जाता है, जैसे इमारतों, जहाजों और विमानों के डिज़ाइन। प्लॉटर दो प्रकार के होते हैं: (i) Flat-bed plotter, (ii) Drum Plotter।*

4. **Speaker:** Speakers are used to get sound output from the computer to listen to voices during online meetings, music, and system sounds.
*स्पीकर: Speakers का उपयोग कंप्यूटर से sound output प्राप्त करने के लिए किया जाता है ताकि ऑनलाइन मीटिंग के दौरान आवाज़, संगीत और सिस्टम साउंड सुना जा सके।*

5. **Projector:** A projector is an output device that presents images onto a large projection screen to show photos, videos and presentations. There are three types: (i) CRT, (ii) LCD, (iii) DLP Projectors.
*प्रोजेक्टर: Projector एक output device है जो बड़ी स्क्रीन पर images को प्रोजेक्ट करता है ताकि फोटो, वीडियो और presentations को दिखाया जा सके। यह तीन प्रकार के होते हैं: (i) CRT, (ii) LCD, (iii) DLP Projectors।*
